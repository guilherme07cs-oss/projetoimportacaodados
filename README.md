# Projeto Importacao Dados com Python

Este projeto foi feito utilizando a linguagem python, que tem como objetivo exemplificar como seria uma migração de dados entre sistemas. O software consiste em coletar dados de um arquivo .CSV e transferi-lo para um banco de dados, no caso o SQLite. 

## Requisitos

- Python 3.13.7
- SQLite 3.13.1
- Pandas 3.0.3

## Como utilizar 

- Baixe o repositório

```
git clone https://github.com/guilherme07cs-oss/projetoimportacaodados.git
```

-  Coloque o arquivo `.csv` desejado no mesmo diretório do projeto

- No diretório rode:

```
python app.py
```
